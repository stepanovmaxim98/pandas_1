# pandas_1
# Задания по работе с библиотекой pandas

### 1. Создание DataFrame
Создайте `DataFrame` с данными о студентах: имя, возраст, средний балл.

### 2. Загрузка данных
Загрузите CSV-файл с произвольными данными (например, данные о продажах) в `DataFrame` и выведите первые 5 строк.

### 3. Основная информация
Выведите информацию о `DataFrame`: количество строк и столбцов, типы данных, количество пропущенных значений.

### 4. Доступ к данным
Выведите первые 3 строки, последние 5 строк и 10 случайных строк из `DataFrame`.

### 5. Выбор столбцов
Выведите только определенные столбцы (например, имя и средний балл) из `DataFrame`.

### 6. Фильтрация данных
Выберите всех студентов старше 20 лет и с оценками выше 4.5.

### 7. Добавление нового столбца
Добавьте столбец с результатом "Зачет" или "Не зачет" в зависимости от среднего балла (>3 — зачет, иначе не зачет).

### 8. Изменение данных в столбце
Умножьте все средние баллы на 1.1 и округлите до 2 знаков после запятой.

### 9. Удаление строк и столбцов
Удалите столбец с возрастом и выведите измененный `DataFrame`.

### 10. Группировка данных
Группируйте студентов по возрасту и посчитайте средний балл в каждой группе.

### 11. Сортировка данных
Отсортируйте `DataFrame` по среднему баллу в порядке убывания.

### 12. Агрегирование данных
Найдите минимальный, максимальный и средний балл среди студентов.

### 13. Заполнение пропущенных данных
Замените пропущенные значения в `DataFrame` на среднее значение соответствующего столбца.

### 14. Объединение DataFrame
Создайте два `DataFrame` (например, с разными группами студентов) и объедините их.

### 15. Сохранение данных
Сохраните обработанный `DataFrame` в CSV-файл.
